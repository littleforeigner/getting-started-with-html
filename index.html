The Basics of the Web & HTML

The Web

The World Wide Web was invented in early 1990s. It is basically a collection of, at the current time approximately 30 billion, HTML documents. Web pages are HTML documents that are read, interpreted and displayed by browsers on our computers. Hyperlinks - which can link one HTML document to another - contained within these documents are what make “the web weblike”.

Major Components of The Web

  • Your computer running a bowser to display files found on the web
  • The Internet - the world’s largest computer network
  • http - the main protocol of the web via which information is sent between your computer and servers
  • Servers - computers that host the files that make up the web

HTML

HTML is the basis for almost every web page and can be thought of as the “glue” that connects everything together.

Functions of HTML

  • Renders text content we see on web pages.
  • Controls how content is presented via markup text that tells computers what to do in a language they understand.
  • References other documents such as images & video.
  • Connectes web pages via hyperlinks.

Components of HTML markup

HTML elements are made up of tags, attributes and values. Eg in the anchor element Udacity
  • a is the opening tag
  • /a is the closing tag
  • href is an attribute
  • http://www.udacity.com is a value
Void tags such as img and br do not require closing tags

Two kinds of HTML elements

  • Inline - no break up of page flow, do not force new lines before or after placement
    Examples include b, em, a, img & span.
  • Block - preserves whitespace, new lines appear before and after the element
    Examples include p, form, div
